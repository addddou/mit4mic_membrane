# mit4mic_membrane

四传声器薄层材料阻抗管测试系统，基于 LabVIEW 开发，用于薄层/膜类材料声学参数测试、四通道麦克风采集、校准、传输阻抗计算和结果导出。

## 项目简介

本项目对应“4MIC 薄层材料测试系统 V1.0”，主工程文件为 `mit4mic_new.lvproj`，主程序入口为 `VIs/main.vi`。项目通过 4 个传声器通道采集阻抗管内声压信号，结合样品几何参数、麦克风位置、校准数据和激励信号设置，完成薄层材料相关声学量的计算与显示。

项目中已包含操作手册、算法推导材料、示意图、WaveIO 声卡接口库以及测试/校准数据示例。

## 主要功能

- 四通道麦克风数据采集与校准
- ASIO/WASAPI 声卡输入输出调用
- 激励信号参数设置
- 基本管体参数和样品参数配置
- 传递函数与复数频响处理
- 反射系数、修正传输波数、传输阻抗计算
- 低频、中高频传输阻抗计算与拼接
- 流速、传输阻抗等结果导出
- 操作手册 PDF 打开与辅助说明

## 环境要求

- Windows 系统
- LabVIEW 2021 或兼容版本
- NI Sound and Vibration / Analysis 相关库
- WaveIO 1.08
- 支持 ASIO 或 WASAPI 的音频设备/声卡

如果使用 64 位 LabVIEW，请按照 `waveio_108/README.txt` 的说明，将 `WaveIO_x64.dll` 重命名或配置为项目使用的 `WaveIO.dll`。

## 快速开始

1. 使用 LabVIEW 打开 `mit4mic_new.lvproj`。
2. 确认 `waveio_108` 目录中的 WaveIO 库和 DLL 可以被项目正确加载。
3. 打开并运行 `VIs/main.vi`。
4. 在参数设置中配置阻抗管内径、麦克风位置、样品面积、温度和麦克风灵敏度等参数。
5. 在激励信号界面配置幅值、信号时长和频率范围。
6. 先执行麦克风校准，再进行四通道采集和阻抗计算。
7. 根据需要导出 `传输阻抗Z.csv`、`流速V.csv` 等结果文件。

## 构建可执行程序

LabVIEW 项目中包含构建规格：

- 构建名称：`阻抗管测试系统20241223`
- 目标程序：`阻抗管测试系统V2.exe`
- 版本：`1.0.10`
- 输出目录：`../builds/NI_AB_PROJECTNAME/阻抗管测试系统20241223`

在 LabVIEW Project Explorer 中打开 `程序生成规范`，选择 `阻抗管测试系统20241223` 并执行 Build 即可生成可执行程序。

## 数据文件说明

- `parameter.dat`：保存系统基本参数、管体几何参数、麦克风位置、样品面积和灵敏度等信息。
- `stimulus.dat`：保存激励信号设置，例如幅值、时长、频率上限和频率下限。
- `main.dat`：保存主界面相关状态，例如平滑和绘图数据选择。
- `！校准/MIC*.txt`：四个麦克风通道的校准采样数据。
- `！采集/MIC*.txt`：四个麦克风通道的测试采样数据。
- `传输阻抗Z.csv`、`流速V.csv`：计算结果导出文件示例。

## 注意事项

- 项目路径中包含中文文件名和目录名，请确保 Git、LabVIEW 和系统编码环境可以正确处理 UTF-8/中文路径。
- 首次打开项目后，建议对 `waveio_108` 目录执行 LabVIEW Mass Compile，以适配当前 LabVIEW 版本。
- 若更换声卡或 LabVIEW 位数，请检查 WaveIO DLL 版本是否匹配。
- 采集和校准数据文件可能较大，正式测试前建议按实验批次备份原始数据。
